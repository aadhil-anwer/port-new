Aadhil Anwar

Security engineer. Breaking things to make them safer.

I'm a security engineer working in network security at GoDaddy — infrastructure security, access control architecture, and cloud security on AWS. I spend my days thinking about how networks break and how to make sure they don't.

I've reported high-severity vulnerabilities (CVSS 9.1, 8.8) in production APIs, designed ACL and VLAN segmentation policies for production environments, and integrated security into DevSecOps workflows. I like going deep — understanding systems at the level where the actual problems live.

Outside of work, I spend time on security research, building tools, and testing attack surfaces. I share what I learn along the way.
