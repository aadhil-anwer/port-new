Hey, I am Aadhil

security, networks, and breaking things. always learning.

I'm a security engineer working in network security at GoDaddy, focused on vulnerability research, API security, network defense, and offensive testing.

I've reported high-severity CVEs in production APIs, built vulnerable lab environments for hands-on training, and written servers from raw TCP sockets to understand what's happening at the wire level. Previously at LPU as a Network Engineering Intern and Cybersecurity Lead at the Cisco CyberSec Symposium, where I delivered workshops to 2000+ students.

On the side, I dig into web and API exploitation, read everything I can around systems and security, and share what I learn along the way.
